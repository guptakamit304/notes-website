JavaScript Arrays

Working with Lists in JavaScript

Array Definition

Arrays are data structures that hold multiple values in a single variable, stored in a sequential order, zero-indexed. Elements are accessed via indices starting from zero. They can store various data types and have built-in methods for manipulation.

Array Basics

  • Definition: Arrays are list-like objects that store multiple values in a single variable.
  • Accessing Elements:
    • Use indexes (0-based): arr[0] is the first element.
    • Use arr.length to get the array's length.
  • Common Methods:
    • Adding:
      • arr.push(element) - Adds to the end.
      • arr.unshift(element) - Adds to the start.
    • Removing:
      • arr.pop() - Removes the last element.
      • arr.shift() - Removes the first element.

Array Methods

  • arr.includes(value) - Checks if an array contains a specific value.
  • arr.indexOf(value) - Returns the index of the first occurrence.
  • arr.slice(start, end) - Extracts a portion of the array.
  • arr.splice(start, deleteCount, item) - Adds/removes elements.

Important Notes

  • Spread Syntax: Use the spread syntax (...) to copy arrays or concatenate them. Example: const newArray = [...array1, ...array2]
  • Array Destructuring: Extract values into variables using array destructuring. Example: const [first, second] = array;
  • Immutability: Avoid modifying arrays directly when possible, as it might lead to unexpected bugs. Try to use methods like slice or spread syntax to create new arrays with the changes you want.
