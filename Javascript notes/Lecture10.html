JavaScript Date Object

Working with Dates and Times in JavaScript

Basic Date Methods

  • Date.now(): Returns the current timestamp in milliseconds. 
    console.log(Date.now());
  • Date(): Creates a new Date object with the current date and time or a specific date. 
    let now = new Date();
console.log(now);

let specificDate = new Date('2024-07-25');
console.log(specificDate);

Getting Date Components

  • getDate(): Returns the day of the month for a specified date. 
    let now = new Date();
console.log(now.getDate()); // Output: Day of the month
  • getMonth(): Returns the month (0-11) for a specified date. 
    let now = new Date();
console.log(now.getMonth()); // Output: Current month (0-11)

Getting Year and Time

  • getFullYear(): Returns the year of a specific date. 
    let now = new Date();
console.log(now.getFullYear()); // Output: Current year
  • getHours(): Returns the hour (0-23) for a specified date. 
    let now = new Date();
console.log(now.getHours()); // Output: Current hour

Getting Minutes and Seconds

  • getMinutes(): Returns the minutes (0-59) for a specified date. 
    let now = new Date();
console.log(now.getMinutes()); // Output: Current minutes
  • getSeconds(): Returns the seconds (0-59) for a specified date. 
    let now = new Date();
console.log(now.getSeconds()); // Output: Current seconds

Converting to Strings

  • toDateString(): Converts a date object to a human-readable string. 
    let now = new Date();
console.log(now.toDateString()); // Output: Fri Jul 25 2024
  • toTimeString(): Converts a date object to a string representing the time. 
    let now = new Date();
console.log(now.toTimeString());
// Output: 14:30:15 GMT+0200 (Central European Summer Time)

Important Notes

  • Month Index: The getMonth() method returns a zero-based index, so January is 0, February is 1 and so on.
  • Time Zones: JavaScript dates are based on the user's time zone, but can be manipulated with other methods to get the correct times.
  • Date Formatting Libraries: For more complex date manipulations, it is best to use libraries such as Date-fns and Moment.js.
