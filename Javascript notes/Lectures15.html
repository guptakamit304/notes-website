JavaScript Objects

Understanding and Working with Objects in JavaScript

What is an object in JavaScript?

An object in JavaScript is a collection of properties, where each property is defined as a key-value pair. Objects can also contain methods (functions attached to the object), making them powerful tools for modeling real-world entities and organizing code.

Key Features

  • Objects group related data and behavior into a single structure.
  • They allow dynamic property assignment and access.
  • Objects are used to represent real-world entities like user, car or product.

Creating Objects in JavaScript

  • Using object literals 
    const person = {
    name: "Aayushi",
    age: 15,
    greet: function() {
        console.log(`Hello, my name is ${this.name}`);
    }
};

Using Object Constructor

const car = new Object();
car.make = "Toyota";
car.model = "Camry";
car.drive = function() {
  console.log(`${this.make} ${this.model} is driving`);
};
car.drive(); // Call the method

Using Object.create()

const animal = {
    eat: function() {
        console.log(`${this.name} is eating`);
    }
};

const dog = Object.create(animal);
dog.name = "Buddy";
dog.eat(); // Calls the `eat` method from the prototype

Dot Notation

  • Use the dot (.) operator to access properties or call methods.
  • console.log(person.name); // Access property
person.greet(); // Call method

Bracket Notation

  • Use bracket ([]) for dynamic property access or when keys are not valid identifiers.
  • const property = "age";
console.log(person[property]);

Important Notes

  • Object Properties: JavaScript Object properties can hold any valid data type, including other objects or arrays.
  • 'this' Keyword: When using methods in objects, 'this' keyword can be used to refer to the current object.
  • Object Iteration: There are multiple ways to iterate through the properties of an object, for example: for...in loop can be used for it.
