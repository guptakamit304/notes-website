Control Statements

Controlling the Flow of Your JavaScript Code

Programming Constructs

  • Decision-Making: Conditional statements (if, if-else, else if, switch).
  • Repetitive Execution: Looping statements (for, while, do-while).
  • Handling Flow: Managing the sequence of operations.
  • Enhancing Logic: Structuring logic and iterating through data.

If Statement

Executes code if a condition is true.

if (condition) {
  // Code
}

If-Else Statement

Executes one block if true, another if false.

if (condition) {
  // Code if true
} else {
  // Code if false
}

Else If Statement

Tests new conditions if previous are false.

if (condition1) {
  // Code if true
} else if (condition2) {
  // Code if condition2 is true
} else {
  // Code if all false
}

Switch Statement

Matches an expression to a case and executes statements.

switch (expression) {
  case value1:
    // Code if expression === value1
    break;
  case value2:
    // Code if expression === value2
    break;
  default:
    // Default code
}

For Loop

Executes code a specified number of times.

for (initialization; condition; iteration) {
  // Code
}

While Loop

Executes while a condition is true.

while (condition) {
  // Code
}

Do-While Loop

Executes code once, then repeats if a condition is true.

do {
  // Code
} while (condition);

Important Notes

  • Break Statements: The break statement is used to terminate a loop and can be used inside a `switch` statement, `for` loop, `while` loop and `do while` loop.
  • Continue Statements: The continue statement is used to skip an iteration in a loop and continues with the next iteration, skipping the remaining code inside the loop.
  • Infinite Loops: Be careful while creating loops. If the condition in a while or do-while loop is always true, it will create an infinite loop and crash your browser/program.
