JavaScript Operators

A Comprehensive Guide to JavaScript Operators

JavaScript operators perform operations on variables and values.

Arithmetic Operators

  • + : Addition (a + b)
  • - : Subtraction (a - b)
  • * : Multiplication (a * b)
  • / : Division (a / b)
  • % : Modulus (Remainder) (a % b)
  • ** : Exponentiation (a ** b)
  • ++ : Increment (a++ or ++a)
  • -- : Decrement (a-- or --a)

Assignment Operators

  • = : Assign (a = 5)
  • += : Add and assign (a += 5)
  • -= : Subtract and assign (a -= 5)
  • *= : Multiply and assign (a *= 5)
  • /= : Divide and assign (a /= 5)
  • %= : Modulus and assign (a %= 5)

Comparison Operators

  • == : Equal to (a == b)
  • === : Strict equal to (value and type) (a === b)
  • != : Not equal to (a != b)
  • !== : Strict not equal to (a !== b)
  • > : Greater than (a > b)
  • < : Less than (a < b)
  • >= : Greater than or equal to (a >= b)
  • <= : Less than or equal to (a <= b)

Logical Operators

  • && : Logical AND (a && b)
  • || : Logical OR (a || b)
  • ! : Logical NOT (!a)

Bitwise Operators

  • & : AND
  • | : OR
  • ^ : XOR
  • ~ : NOT
  • << : Left shift
  • >> : Right shift
  • >>> : Zero-fill right shift

Ternary Operator

  • condition ? expr1 : expr2

Important Notes

  • Strict Equality: Always prefer === and !== over == and != for comparison. The strict operators check both value and type.
  • Type Coercion: Be aware that JavaScript sometimes converts values to different types. Example: "5" == 5 is true, but "5" === 5 is false.
  • Short-Circuiting: The logical AND (&&) and OR (||) operators can perform short-circuiting which can sometimes impact your code's flow.
