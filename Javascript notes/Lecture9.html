JavaScript Built-in Functions

A Comprehensive Guide to JavaScript's Core Functions

JavaScript provides numerous built-in functions for common tasks. These functions are part of the core language and don't require additional libraries to be used.

String Functions

  1. charAt(index)

    Returns the character at the specified index in a string.

    let str = "Hello"; console.log(str.charAt(1)); // Output: "e"

  2. concat(string1, string2, ...)

    Concatenates two or more strings.

    let str1 = "Hello"; let str2 = "World"; console.log(str1.concat(" ", str2)); // Output: "Hello World"

  3. includes(substring)

    Checks if a string contains a specified substring.

    let str = "JavaScript is fun"; console.log(str.includes("fun")); // Output: true

  4. indexOf(substring)

    Returns the index of the first occurrence of a substring. Returns -1 if not found.

    let str = "Learn JavaScript"; console.log(str.indexOf("Java")); // Output: 6

  5. lastIndexOf(substring)

    Returns the index of the last occurrence of a substring.

    let str = "JavaScript is JavaScript"; console.log(str.lastIndexOf("Script")); // Output: 20

  6. slice(start, end)

    Extracts part of a string and returns it as a new string.

    let str = "Hello, World!"; console.log(str.slice(0, 5)); // Output: "Hello"

  7. substring(start, end)

    Extracts characters from a string between two indices. Does not support negative indices.

    let str = "JavaScript"; console.log(str.substring(4, 10)); // Output: "Script"

  8. substr(start, length)

    Extracts part of a string, starting at a specified index, and with a specified length. (Deprecated)

    let str = "JavaScript"; console.log(str.substr(4, 6)); // Output: "Script"

  9. toLowerCase()

    Converts a string to lowercase.

    let str = "HELLO WORLD"; console.log(str.toLowerCase()); // Output: "hello world"

  10. toUpperCase()

    Converts a string to uppercase.

    let str = "hello world"; console.log(str.toUpperCase()); // Output: "HELLO WORLD"

  11. trim()

    Removes whitespace from both ends of a string.

    let str = " Hello, World! "; console.log(str.trim()); // Output: "Hello, World!"

  12. split(separator)

    Splits a string into an array of substrings based on a specified separator.

    let str = "a,b,c"; console.log(str.split(",")); // Output: ["a", "b", "c"]

  13. replace(searchValue, replaceValue)

    Replaces the first occurrence of a substring with another substring.

    let str = "Hello World"; console.log(str.replace("World", "JavaScript")); // Output: "Hello JavaScript"

  14. replaceAll(searchValue, replaceValue)

    Replaces all occurrences of a substring with another substring.

    let str = "abcabcabc"; console.log(str.replaceAll("abc", "xyz")); // Output: "xyzxyzxyz"

  15. startsWith(substring)

    Checks if a string starts with a specified substring.

    let str = "JavaScript"; console.log(str.startsWith("Java")); // Output: true

  16. endsWith(substring)

    Checks if a string ends with a specified substring.

    let str = "JavaScript"; console.log(str.endsWith("Script")); // Output: true

  17. repeat(count)

    Repeats a string a specified number of times.

    let str = "JS"; console.log(str.repeat(3)); // Output: "JSJSJS"

Number Functions

  1. toString()

    Converts a number to a string.

    let num = 123; console.log(num.toString()); // Output: "123"

  2. toFixed(digits)

    Formats a number to a specified number of decimal places and returns it as a string.

    let num = 123.456; console.log(num.toFixed(2)); // Output: "123.46"

  3. toExponential(fractionDigits)

    Returns the number in exponential notation with a specified number of digits after the decimal point.

    let num = 12345; console.log(num.toExponential(2)); // Output: "1.23e+4"

  4. toPrecision(digits)

    Formats a number to a specified length and returns it as a string.

    let num = 123.456; console.log(num.toPrecision(4)); // Output: "123.5"

  5. valueOf()

    Returns the primitive value of a Number object.

    let num = new Number(123); console.log(num.valueOf()); // Output: 123

  6. isFinite(number)

    Checks if the value is a finite number. Returns true or false.

    console.log(Number.isFinite(123)); // Output: true console.log(Number.isFinite(Infinity)); // Output: false

  7. isInteger(number)

    Checks if the value is an integer. Returns true or false.

    console.log(Number.isInteger(123)); // Output: true console.log(Number.isInteger(123.45)); // Output: false

  8. isNaN(number)

    Checks if the value is NaN (Not-a-Number). Returns true or false.

    console.log(Number.isNaN(123)); // Output: false console.log(Number.isNaN(NaN)); // Output: true

  9. parseInt(string)

    Parses a string and returns an integer. Stops parsing at the first non-digit character.

    console.log(Number.parseInt("123abc")); // Output: 123

  10. parseFloat(string)

    Parses a string and returns a floating-point number. Stops parsing at the first non-digit character.

    console.log(Number.parseFloat("123.45abc")); // Output: 123.45

  11. MAX_VALUE

    Returns the largest representable number in JavaScript.

    console.log(Number.MAX_VALUE); // Output: 1.7976931348623157e+308

  12. MIN_VALUE

    Returns the smallest positive representable number in JavaScript.

    console.log(Number.MIN_VALUE); // Output: 5e-324

  13. POSITIVE_INFINITY

    Represents infinity in JavaScript.

    console.log(Number.POSITIVE_INFINITY); // Output: Infinity

  14. NEGATIVE_INFINITY

    Represents negative infinity in JavaScript.

    console.log(Number.NEGATIVE_INFINITY); // Output: -Infinity

  15. isSafeInteger(number)

    Checks if the value is a safe integer (within the range of -(2^53 - 1) to (2^53 - 1)).

    console.log(Number.isSafeInteger(123)); // Output: true console.log(Number.isSafeInteger(1e20)); // Output: false

  16. EPSILON

    Returns the smallest difference between two representable numbers.

    console.log(Number.EPSILON); // Output: 2.220446049250313e-16

Math Functions

  1. Math.abs(x)

    Returns the absolute (positive) value of a number.

    console.log(Math.abs(-5)); // Output: 5

  2. Math.ceil(x)

    Rounds a number up to the nearest integer.

    console.log(Math.ceil(4.2)); // Output: 5

  3. Math.floor(x)

    Rounds a number down to the nearest integer.

    console.log(Math.floor(4.8)); // Output: 4

  4. Math.round(x)

    Rounds a number to the nearest integer.

    console.log(Math.round(4.5)); // Output: 5

  5. Math.trunc(x)

    Returns the integer part of a number, removing any fractional digits.

    console.log(Math.trunc(4.9)); // Output: 4

  6. Math.max(a, b, ...)

    Returns the largest value among the given numbers.

    console.log(Math.max(3, 5, 1)); // Output: 5

  7. Math.min(a, b, ...)

    Returns the smallest value among the given numbers.

    console.log(Math.min(3, 5, 1)); // Output: 1

  8. Math.pow(base, exponent)

    Returns the base raised to the power of the exponent.

    console.log(Math.pow(2, 3)); // Output: 8

  9. Math.sqrt(x)

    Returns the square root of a number.

    console.log(Math.sqrt(16)); // Output: 4

  10. Math.cbrt(x)

    Returns the cube root of a number.

    console.log(Math.cbrt(27)); // Output: 3

  11. Math.random()

    Returns a random number between 0 (inclusive) and 1 (exclusive).

    console.log(Math.random()); // Output: (random value between 0 and 1)

  12. Math.PI

    Returns the value of Ï€ (pi).

    console.log(Math.PI); // Output: 3.141592653589793

  13. Math.E

    Returns the value of Euler's constant (e).

    console.log(Math.E); // Output: 2.718281828459045

  14. Math.log(x)

    Returns the natural logarithm (base e) of a number.

    console.log(Math.log(Math.E)); // Output: 1

  15. Math.log10(x)

    Returns the base-10 logarithm of a number.

    console.log(Math.log10(100)); // Output: 2

  16. Math.log2(x)

    Returns the base-2 logarithm of a number.

    console.log(Math.log2(8)); // Output: 3

  17. Math.exp(x)

    Returns e raised to the power of x.

    console.log(Math.exp(1)); // Output: 2.718281828459045

  18. Math.sin(x)

    Returns the sine of an angle (in radians).

    console.log(Math.sin(Math.PI / 2)); // Output: 1

  19. Math.cos(x)

    Returns the cosine of an angle (in radians).

    console.log(Math.cos(Math.PI)); // Output: -1

  20. Math.tan(x)

    Returns the tangent of an angle (in radians).

    console.log(Math.tan(Math.PI / 4)); // Output: 1

  21. Math.asin(x)

    Returns the arcsine (in radians) of a number.

    console.log(Math.asin(1)); // Output: 1.5707963267948966

  22. Math.acos(x)

    Returns the arccosine (in radians) of a number.

    console.log(Math.acos(0)); // Output: 1.5707963267948966

  23. Math.atan(x)

    Returns the arctangent (in radians) of a number.

    console.log(Math.atan(1)); // Output: 0.7853981633974483

Important Notes

  • String Immutability: String methods do not change original string but create and return new ones.
  • Number Precision: Be aware of potential precision issues when performing math operations on floating point numbers.
  • Math Constants: Math.PI and Math.E are constants not functions, thus do not require ().
