HTML Forms Overview

The <form> Element

The HTML <form> element is used to create a form for user input. It acts as a container for various form elements like text fields, checkboxes, radio buttons, and submit buttons.

Example of a basic form structure:

<form>
    ...form elements...
</form>

Forms are essential for collecting user data, such as login credentials, survey responses, or file uploads.

HTML Form Attributes

The <form> element has several attributes that define its behavior:

  • Action: Specifies the URL where form data is sent when submitted. For example: 
    <form action="action_page.php">
    <input type="text" name="name">
    <button type="submit">Submit</button>
</form>
  • Method: Determines how form data is sent. Common values:
    • GET: Appends form data to the URL.
    • POST: Sends form data as a request body.
  • Target: Specifies where to display the response. Values include:
    • _self: Opens in the same tab (default).
    • _blank: Opens in a new tab or window.

HTML Form Elements

The <form> element can contain various child elements to collect data:

  • <input>: Defines input fields (text, password, etc.).
  • <label>: Associates labels with input elements for better accessibility.
  • <select>: Creates dropdown menus.
  • <textarea>: Defines multi-line text input fields.
  • <button>: Creates clickable buttons.
  • <fieldset> and <legend>: Group related form elements.
  • <datalist>: Provides autocomplete options for input fields.
  • <output>: Displays calculated results.
  • <option> and <optgroup>: Define options within dropdowns.

Each of these elements serves a specific purpose and helps in building interactive and user-friendly forms.
