HTML Overview

What is HTML?

- HTML stands for Hyper Text Markup Language.
- It is the standard language for creating web pages.
- Describes the structure of web pages using elements.
- Elements define content like headings, paragraphs, links, etc.

Basic HTML Structure

<html>
    <head>
        <title>Introduction</title>
    </head>
    <body>
        <p>This is an example</p>
    </body>
</html>

HTML Head Tag

The <head> tag contains meta information about the document, such as the title, character encoding, linked stylesheets, and scripts.

HTML Body Tag

The <body> tag contains the content displayed on the webpage, such as text, images, and other elements.
