Definition and Usage

The <table> tag defines an HTML table.

An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements.

- The <tr> element defines a table row.
- The <th> element defines a table header.
- The <td> element defines a table cell.

Additional elements like <caption>, <colgroup>, <thead>, <tfoot>, and <tbody> can enhance table functionality.

Defining Tables in HTML

Tables in HTML are defined with the <table> tag. Rows are defined using the <tr> tag, headers with <th>, and data cells with <td>.

By default, table headings are bold and centered. Data cells are used to hold table content.

Syntax:
<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
HTML Tags Descriptions
<table> Defines the structure for organizing data in rows and columns.
<tr> Represents a row within an HTML table.
<th> Defines a table header cell for titles or headings.
<td> Defines a standard data cell holding content or data.
<caption> Provides a title or description for the entire table.
<thead> Defines the header section of a table.
<tbody> Represents the main content area of a table.
<tfoot> Specifies the footer section of a table.
<col> Defines attributes for individual table columns.
<colgroup> Groups a set of columns in a table for shared formatting.
