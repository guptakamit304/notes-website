HTML Links - Hyperlinks

HTML links are hyperlinks.

You can click on a link and jump to another document.
When you move the mouse over a link, the mouse arrow will turn into a little hand.
Note: A link does not have to be text. It can be an image or any other HTML element!

Syntax
The HTML <a> tag defines a hyperlink. Its syntax is:

<a href="url">Link Text</a>

Attributes of <a> Tag:
- href: Specifies the URL of the page the link goes to.
- target: Specifies where to open the link (e.g., _blank for new tab).
- title: Provides additional information on hover.
- download: Specifies a file to be downloaded when the link is clicked.
