Images and Videos in HTML

Images

HTML Images Syntax

The HTML <img> tag is used to embed images in a web page. Images are not directly inserted into the page but are linked using the src attribute, which specifies the image source.

Key Points:
- The <img> tag is empty (does not have a closing tag).
- It requires the src attribute to specify the image path.
- The alt attribute provides alternative text if the image cannot load.

Example Syntax

<img src="url" alt="Alternate Text">

Image Size - Width and Height

You can define the size of an image using the style attribute:

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

Summary

  • Use <img> to define an image.
  • src specifies the image URL.
  • alt provides alternate text if the image fails to load.
  • Define size using width and height attributes or CSS properties.
  • Use the CSS float property to position images (left or right).

Videos

The HTML <video> Element

The <video> element allows embedding videos in a webpage.

Example Syntax

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    Your browser does not support the video tag.
</video>

Key Attributes:
- controls: Adds playback controls.
- autoplay: Plays the video automatically.
- loop: Repeats the video.
- muted: Starts the video without sound.

HTML Video - Methods, Properties, and Events

The HTML DOM provides methods, properties, and events to control videos:

  • Play and pause videos using play() and pause().
  • Set or get volume using the volume property.
  • Track playback status with events like onplay, onpause, etc.

HTML Video Tags

  • <video>: Defines a video or movie.
  • <source>: Specifies multiple media sources for the video.
  • <track>: Adds captions or subtitles for the video.
