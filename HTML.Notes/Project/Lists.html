HTML Lists

Unordered HTML List

An unordered list is used to group a set of related items in no particular order. It starts with the <ul> tag, and each list item is defined using the <li> tag. By default, list items are marked with bullet points (small black circles).

Example: Unordered List

<!DOCTYPE html>
<html>
<body>
  <ul>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
  </ul>
</body>
</html>

Rendered Output:

  • Coffee
  • Tea
  • Milk

Ordered HTML List

An ordered list is used to group a set of related items in a specific order. It starts with the <ol> tag, and each list item is defined using the <li> tag. By default, list items are numbered (e.g., 1, 2, 3).

Example: Ordered List

<!DOCTYPE html>
<html>
<body>
  <ol>
    <li>Tea</li>
    <li>Milk</li>
  </ol>
</body>
</html>

Rendered Output:

  1. Tea
  2. Milk
