Bootstrap Components Notes

Badges

Badges are used to add a small numeric or symbolic indicator on a UI element, often used for notifications or counts.

<span class="badge bg-primary">New</span>

Example: <span class="badge bg-primary">New</span> will create a blue badge with the text "New".

Progress Bars

Progress bars are used to indicate the progress of a task or process.

<div class="progress">
    <div class="progress-bar" style="width: 50%" role="progressbar"></div>
</div>

Example: <div class="progress"> <div class="progress-bar" style="width: 50%" role="progressbar"></div> </div> will create a progress bar filled halfway.

Navigation

Navigation components are used to create menus, tabs, and links to help users navigate through a page or app.

Navigation Tabs

Turn a basic navigation bar into tabs by adding the .nav-tabs class.

<ul class="nav nav-tabs">
    <li class="nav-item">
        <a class="nav-link active">Home</a>
    </li>
</ul>

Navigation Pills

Pills are a variation of navigation tabs with rounded corners. Use the .nav-pills class.

<ul class="nav nav-pills">
    <li class="nav-item">
        <a class="nav-link active">Profile</a>
    </li>
</ul>

Navigation Bar

A navigation bar is a component that is typically fixed at the top of the page. It contains links for navigation purposes.

<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <a class="navbar-brand" href="#">Navbar</a>
    <div class="collapse navbar-collapse">
        <ul class="navbar-nav">
            <li class="nav-item">
                <a class="nav-link">Home</a>
            </li>
        </ul>
    </div>
</nav>

Centered Navbar

Add the .justify-content-center class to the navbar to center its contents.

<nav class="navbar navbar-expand-lg navbar-light bg-light justify-content-center">
    <a class="navbar-brand" href="#">Centered Navbar</a>
</nav>
