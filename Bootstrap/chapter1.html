Chapter 1: Bootstrap 5 Overview

A comprehensive guide to mastering Bootstrap 5 for modern web development.

Typography

Typography in Bootstrap 5 is built around a default font size of 1rem (16px by default) and a line-height of 1.5, ensuring excellent readability across devices. Additionally, all paragraph elements <p> have consistent spacing with no top margin and a bottom margin of 1rem.

The heading elements (<h1> to <h6>) are styled with bold font weights and responsive font sizes. This means they adjust automatically based on the deviceâ€™s screen size to provide optimal readability.

Text Colors

Bootstrap 5 includes a variety of contextual text color classes to convey meaning and style. These classes make it easy to apply colors consistently across your application.

The available classes are:

  • .text-primary - Represents primary theme color.
  • .text-success - Indicates success or positive messages.
  • .text-info - Highlights informational content.
  • .text-warning - Warns about potential issues.
  • .text-danger - Signifies errors or critical messages.
  • .text-muted - De-emphasizes text.

Background Colors

Background color classes allow you to easily style the background of elements. Use the .bg-* classes to apply colors like:

  • .bg-primary - A primary color background.
  • .bg-danger - A danger color background for alerts.
  • .bg-warning - A yellow warning background.
  • .bg-light - A light grey background.
  • .bg-dark - A dark background for contrast.

Tip: Combine background and text color classes (e.g., .bg-primary .text-white) for a complete styling solution.

Tables

Bootstrap 5 tables are lightweight and come with additional features for styling and responsiveness.

  • Basic Table: Add the .table class for a simple, clean table.
  • Responsive Table: Wrap the table in a .table-responsive container to enable horizontal scrolling on smaller devices.
  • Contextual Classes: Use classes like .table-primary, .table-danger, or .table-success to add contextual styling to rows or cells.
