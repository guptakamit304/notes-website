CSS Selectors

CSS selectors are used to select elements on a web page in order to apply styles. Below, you'll learn about different types of selectors in CSS, which are used to target specific HTML elements based on various attributes.

1. Element Selectors

Element selectors target HTML elements based on their tag names. This is the most common type of selector used to apply styles to elements like paragraphs, headings, or divs.

p {
    color: blue;
}

Example: p { color: blue; } will turn all <p> elements blue.

2. Class Selectors

Class selectors are used to select elements based on the class attribute. A class selector is denoted by a dot (.) followed by the class name.

.highlight {
    background-color: yellow;
}

Example: .highlight { background-color: yellow; } will apply a yellow background to any element with the class highlight.

3. ID Selectors

ID selectors are used to select a single element based on its ID attribute. An ID selector is denoted by a hash symbol (#) followed by the ID name. The ID must be unique within the document.

#unique {
    font-size: 30px;
}

Example: #unique { font-size: 30px; } will change the font size of the element with the ID unique to 30px.

4. Attribute Selectors

Attribute selectors are used to select elements based on the presence or value of an attribute. You can use them to target elements with specific attributes or specific attribute values.

[type="text"] {
    border: 1px solid #000;
}

Example: [type="text"] { border: 1px solid #000; } will add a black border to all input elements with a type attribute of "text".

5. Other Common Selectors

There are other selectors as well, such as:

  • Universal Selector: * - Selects all elements on the page.
  • Descendant Selector: parent descendant - Selects elements that are nested inside a parent element.
  • Child Selector: parent > child - Selects direct children of a parent element.
  • Group Selector: selector1, selector2 - Groups multiple selectors together, so they share the same style.

Selectors in CSS allow you to target specific HTML elements to apply styles, and with the various selector types, you have powerful ways to style and control your page's design.
