CSS Multiple Backgrounds

CSS allows you to add multiple background images for an element, through the background-image property.

The different background images are separated by commas, and the images are stacked on top of each other, where the first image is closest to the viewer.

For example, you can layer a pattern and an image together, creating a more complex background effect.

Note: You can also define the position and repetition of each background image separately using properties like background-position and background-repeat.

Example Code
#example1 {
  background-image: url('img_flwr.gif'), url('paper.gif');
  background-position: right bottom, left top;
  background-repeat: no-repeat, repeat;
}

In this example, the first background image (flower) will be placed at the bottom-right corner, and the second background image (paper) will be placed at the top-left corner. The flower image won't repeat, while the paper image will repeat to fill the element.

The background-image property can accept multiple URLs, and each URL will apply to a layer on the element. The layers are stacked from top to bottom in the order they are listed.

Besides images, you can also use gradients as background images. This allows for more advanced visual effects like gradient overlays or background transitions.

If you're using this technique on a website, it's important to ensure that the background images don't interfere with the readability of the content. Consider using background-size and background-attachment to fine-tune how backgrounds behave on different devices.
