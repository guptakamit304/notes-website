Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the look and feel of the webpage, such as colors, fonts, spacing, and positioning. Here are some key features and benefits of using CSS:

  • Separation of concerns: Keeps content (HTML) and presentation (CSS) separate.
  • Consistency: Ensures a consistent design across multiple pages.
  • Flexibility and control: Allows fine-grained control over the appearance of elements.
  • Easy to manage: CSS makes it easier to update and maintain the styling of the entire website.

CSS Syntax

The basic syntax of CSS is:

selector {
  property: value;
}

Example:

h1 {
  color: blue;
  font-size: 24px;
}

This sets the text color of <h1> elements to blue and the font size to 24px.

Types of CSS

There are three main types of CSS that you can use to style HTML elements:

  1. Inline CSS: Applied directly to an HTML element using the style attribute.
    2. Eg: <p style="color:blue; font-size:16px;">This is a paragraph</p>
  2. Internal CSS: Defined within the <style> element in the <head> section of the HTML document.
    3. Eg:
<style>
  body {
    background-color: red;
  }
  h1 {
    color: red;
    margin-left: 60px;
  }
</style>
  3. External CSS: Stored in a separate .css file and linked to the HTML document using the <link> element.
    4. Eg (Index.html):
<head>
  <link rel="stylesheet" type="text/css" href="style.css">
</head>

Eg (style.css):
p {
  color: blue;
  font-size: 16px;
}
