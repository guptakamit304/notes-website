What are CSS Animations?

An animation allows an element to gradually change from one style to another. You can modify various CSS properties and define their behavior over time using keyframes.

Keyframes define the intermediate steps for the animation and determine how styles change during the animation.

Note: The animation-duration property specifies how long an animation takes to complete. If this property is not set, the animation will not occur, as the default value is 0 seconds.

Delay an Animation

The animation-delay property introduces a delay before the animation starts.

Example: A 2-second delay before the animation begins.

Animation Directions

The animation-direction property defines whether an animation should run forwards, backwards, or alternate between these directions.

  • normal: The animation plays forwards (default).
  • reverse: The animation plays backwards.
  • alternate: The animation alternates between forwards and backwards.
  • alternate-reverse: The animation alternates, starting backwards first.
