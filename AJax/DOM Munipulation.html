DOM Manipulation

What is DOM (Document Object Model)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of nodes, where each node represents a part of the document (like elements, attributes, text, etc.).

Key Features of JavaScript

  • Platform and Language Independent: The DOM is not tied to any specific programming language. It can be used with JavaScript, Python, or other languages.
  • Dynamic Representation: DOM allows you to interact with and modify the content of a webpage in real-time (dynamic updates).
  • Tree Structure: The DOM represents the document as a tree where each HTML element is a node. This makes it easy to traverse and manipulate.

Uses of DOM:

  • Dynamic HTML Content: The DOM allows developers to create dynamic web pages by manipulating HTML content without reloading the entire page.
  • Event Handling: The DOM enables interaction with elements through events like clicks, mouse movements, and keyboard input.
  • Element Manipulation: You can create, delete, or modify elements on a webpage (e.g., adding a new paragraph, changing the style of a button, etc.).

Key Concepts in DOM Manipulation

Selecting Elements:

To interact with an element in the DOM, we first need to select it. JavaScript provides methods like:

  • document.getElementById(id)
  • document.querySelector(selector)
  • document.getElementsByClassName(class)
