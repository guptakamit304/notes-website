AJAX Notes

Modern Browsers (Fetch API)

  • Modern browsers can use the Fetch API instead of the XMLHttpRequest Object.
  • The Fetch API interface allows browsers to make HTTP requests to web servers.
  • Fetch simplifies AJAX tasks and improves readability.

GET vs POST

GET is simpler and faster but use POST when:

  • Updating a server file or database (avoids caching issues).
  • Sending large amounts of data (POST has no size limit).
  • Handling user input securely.

AJAX Request Overview

  • AJAX enables asynchronous server communication via JavaScript.
  • Data is most commonly exchanged using JSON for compatibility and efficiency.

AJAX Response Formats

  • Text: Plain text or HTML.
  • JSON: Ideal for lightweight data exchanges.
  • XML: Used for structured data.
  • HTML: Response may contain HTML content for insertion.
  • JavaScript: Scripts that execute upon receipt.
